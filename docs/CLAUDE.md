# Kontext für Agenten — Ordner `docs/`

Dieser Ordner beschreibt den **Ist-Zustand** des `example`-Plugins.

Regeln beim Arbeiten hier:
- `docs/` = Ist (wie es jetzt ist). `specs/` = Soll (was es können soll). Nicht
  vermischen.
- Ändert eine Code-Änderung das Verhalten, wird die passende Doc-Datei in
  derselben Änderung mitgepflegt. Veraltete Doku ist schlimmer als keine.
- Einfache Sprache, kurze Sätze, ein Bild (mermaid), wo es hilft.
